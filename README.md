## music-player
## 项目演示 
![image](https://github.com/GitZCY/music-player/blob/master/music.gif)
### 借用node.js版api开发的音乐播放器 api地址:(https://github.com/Binaryify/NeteaseCloudMusicApi)
### clone后进入node.js版api
```cd /NeteaseCloudMusicApi```
### 安装所需插件
```npm i```
### 运行node服务器
```node app.js```

### 再安装本项目所需插件
```npm i```
### 运行项目
```npm run serve```
